§1 — Material Properties (FG-GOEAM)

FG_GOEAM.m · FG_GOEAM_stretch.m

The material consists of two components: Copper (Cu) matrix + Graphene (Gr) reinforcement. The effective properties for each layer are calculated using the Halpin–Tsai model + modification functions.

Material Constants (Values in Code)

SymbolMaterialValueUnit
$E_{Gr}$Graphene elastic modulus929.57 × 10⁹Pa
$\nu_{Gr}$Graphene Poisson ratio0.22—
$\alpha_{Gr}$Graphene thermal expansion coefficient−3.98 × 10⁻⁶1/K
$\rho_{Gr}$Graphene density1800kg/m³
$E_{Cu}$Copper elastic modulus65.79 × 10⁹Pa
$\nu_{Cu}$Copper Poisson ratio0.387—
$\alpha_{Cu}$Copper thermal expansion coefficient16.51 × 10⁻⁶1/K
$\rho_{Cu}$Copper density8800kg/m³
$l_{Gr}$Graphene length (Halpin–Tsai)83.76 × 10⁻¹⁰m
$t_{Gr}$Graphene thickness (Halpin–Tsai)3.4 × 10⁻¹⁰m

Halpin–Tsai Parameters

Geometric parameter and Halpin–Tsai coefficient
\[\xi = \frac{2\,l_{Gr}}{t_{Gr}}, \qquad \eta = \frac{E_{Gr}/E_{Cu} - 1}{E_{Gr}/E_{Cu} + \xi}\]

Modification Functions (Temperature & Volume Fraction Dependent)

Elastic modulus modification function
\[f_E = 1.11 - 1.22\,V_{Gr} - 0.134\,\frac{T}{T_0} + 0.559\,V_{Gr}\frac{T}{T_0} - 5.5\,H_{Gr}V_{Gr} + 38\,H_{Gr}V_{Gr}^2 - 20.6\,H_{Gr}^2 V_{Gr}^2\]
Poisson ratio modification function
\[f_\nu = 1.01 - 1.43\,V_{Gr} + 0.165\,\frac{T}{T_0} - 16.8\,H_{Gr}V_{Gr} - 1.1\,H_{Gr}V_{Gr}\frac{T}{T_0} + 16\,H_{Gr}^2 V_{Gr}^2\]
Thermal expansion modification function
\[f_\alpha = 0.794 - 16.8\,V_{Gr}^2 - 0.0279\left(\frac{T}{T_0}\right)^2 + 0.182\,\frac{T}{T_0}(1 + V_{Gr})\]

Here \(T_0 = 300\) K is the reference temperature, \(H_{Gr}\) is the weight fraction of the hollow graphene, and \(V_{Gr}\) is the local volume fraction of graphene.

Effective Material Properties (For Each Layer)

Elastic modulus (Halpin–Tsai + modification)
\[E = \frac{(1 + \xi\,\eta\,V_{Gr})\,E_{Cu}\,f_E}{1 - \eta\,V_{Gr}}, \qquad V_{Cu} = 1 - V_{Gr}\]
Poisson ratio
\[\nu = \left(\nu_{Gr}\,V_{Gr} + \nu_{Cu}\,V_{Cu}\right)f_\nu\]
Thermal expansion coefficient
\[\alpha = \left(\alpha_{Gr}\,V_{Gr} + \alpha_{Cu}\,V_{Cu}\right)f_\alpha\]

§2 — Graphene Volume Fraction Distribution

GOri_distribution.m · FG_GOEAM.m · FG_GOEAM_stretch.m

The distribution of graphene across the thickness is defined by three different patterns. \(V_{Gr}^s\) denotes the global graphene volume fraction, and \(z_{mid}\) is the mid-plane coordinate of the layer.

Weight Fraction to Volume Fraction Conversion

Global graphene volume fraction
\[V_{Gr}^s = \frac{W_{Gr}}{W_{Gr} + (\rho_{Gr}/\rho_{Cu})(1 - W_{Gr})}\]

Here \(W_{Gr}\) is the graphene weight fraction (e.g., 0.025 = 2.5%).

Three Distribution Patterns (For each layer \(k\), with \(z_{mid}\) being the layer mid-plane)

U-WGr (Uniform)
\[V_{Gr}(z) = V_{Gr}^s\]

Graphene is uniformly distributed across the thickness.

X-WGr (Concentrated at the surfaces)
\[V_{Gr}(z) = 2\,V_{Gr}^s \cdot \frac{2|z_{mid}|}{h}\]

Maximum at the surfaces, zero at the mid-plane.

O-WGr (Concentrated at the mid-plane)
\[V_{Gr}(z) = 2\,V_{Gr}^s \left(1 - \frac{2|z_{mid}|}{h}\right)\]

Maximum at the mid-plane, zero at the surfaces.

Tabaka indeksi: Plaka \(N_L = 10\) tabakaya bölünür. \(k\)-inci tabaka için: \(z_{bot} = (N_L/2-k)\,h/N_L\), \(z_{top} = (N_L/2-k+1)\,h/N_L\), \(z_{mid} = (z_{bot}+z_{top})/2\).

§3 — HSDT: 2D Plane Stress Stiffness Constants \(Q_{ij}\)

FG_GOEAM.m — Standart teori

HSDT (Higher-order Shear Deformation Theory) assumes a plane-stress state: \(\sigma_z = 0\). Under this condition, the effective stiffness constants are:

2D reduced stiffness constants
\[Q_{11} = Q_{22} = \frac{E}{1 - \nu^2}, \qquad Q_{12} = Q_{21} = \frac{E\,\nu}{1-\nu^2}, \qquad Q_{66} = Q_{44} = Q_{55} = \frac{E}{2(1+\nu)}\]
Important: The denominator \(1-\nu^2\) accounts only for the in-plane Poisson coupling (\(x\)–\(y\)). The transverse (\(z\)) Poisson effect is neglected.

Thermal Force Integrand Functions (HSDT)

Thermal load coefficient
\[\beta = (Q_{11} + Q_{12})\,\alpha\,\Delta T, \qquad \Delta T = T - T_0\]
Thermal forces and moments
\[n_x(z)=n_y(z)=\beta, \quad m_x(z)=m_y(z)=\beta\,z, \quad p_x(z)=p_y(z)=\beta\,z^3\]

§4 — Quasi-3D: 3D Elasticity Stiffness Constants \(C_{ij}\) YENİ

FG_GOEAM_stretch.m — Thickness stretching theory

Quasi-3D theory assumes thickness-direction stretching (\(\varepsilon_z \neq 0\)). Therefore, the full 3D elasticity constants are used:

3D elasticity constants
\[C_{11} = C_{22} = C_{33} = \frac{E(1-\nu)}{(1+\nu)(1-2\nu)}, \qquad C_{12} = C_{13} = C_{23} = \frac{E\,\nu}{(1+\nu)(1-2\nu)}\]
Difference from HSDT: The denominator contains \((1+\nu)(1-2\nu)\) instead of \((1-\nu^2)\). This fully accounts for the thickness-direction Poisson effect (the feedback from \(\sigma_z \to \varepsilon_z\)).

Thickness Stretching Thermal Coefficient

Quasi-3D thermal thickness force
\[\beta_z = (2\,C_{12} + C_{11})\,\alpha\,\Delta T, \qquad n_z(z) = \beta_z\]

The thermal force in the thickness direction is used as a correction term in the condensation (see §6).

§5 — Thickness Integrals: \(A,B,D,E,F,H\) Matrices

FG_GOEAM.m · FG_GOEAM_stretch.m

The stiffness matrices are obtained by multiplying the material constants by powers of the thickness coordinate \(z\) and integrating through the thickness. The same integration structure is used in both theories — only the integrand differs.

General integral definition (Eq. 17)
\[\left(A_{ij},\;B_{ij},\;D_{ij},\;E_{ij},\;F_{ij},\;H_{ij}\right) = \int_{-h/2}^{h/2} \bar{Q}_{ij}\,\left(1,\;z,\;z^2,\;z^3,\;z^4,\;z^6\right)dz\]
For HSDT: \(\bar{Q}_{ij} = Q_{ij}(z)\)  |  For Quasi-3D: \(\bar{Q}_{ij} = C_{ij}(z)\) (before condensation, with subscript \({}_{3D}\))

Numerical Integration (Midpoint Rule, NL=10 layers)

A matrix terms (the contribution of each layer is summed)
\[A_{ij} = \sum_{k=1}^{N_L} Q_{ij}^{(k)}\,(z_{top}^{(k)} - z_{bot}^{(k)})\]
B, D, E, F, H matrisleri
\[B_{ij} = \sum_{k=1}^{N_L} Q_{ij}^{(k)}\,\frac{(z_{top}^{(k)})^2-(z_{bot}^{(k)})^2}{2}, \quad D_{ij} = \sum_{k=1}^{N_L} Q_{ij}^{(k)}\,\frac{(z_{top}^{(k)})^3-(z_{bot}^{(k)})^3}{3}\] \[E_{ij} = \sum_{k=1}^{N_L} Q_{ij}^{(k)}\,\frac{(z_{top}^{(k)})^4-(z_{bot}^{(k)})^4}{4}, \quad F_{ij} = \sum_{k=1}^{N_L} Q_{ij}^{(k)}\,\frac{(z_{top}^{(k)})^5-(z_{bot}^{(k)})^5}{5}\] \[H_{ij} = \sum_{k=1}^{N_L} Q_{ij}^{(k)}\,\frac{(z_{top}^{(k)})^7-(z_{bot}^{(k)})^7}{7}\]

Note: The \(H_{ij}\) integral uses \(z^6\); in the numerical integration, the difference of \(z^7/7\) is evaluated.

Shear Stiffness Matrices

Transverse shear terms
\[A_{44}=A_{55}=\sum Q_{44}^{(k)}\,\Delta z_k,\quad D_{44}=D_{55}=\sum Q_{44}^{(k)}\,\frac{\Delta(z^3)_k}{3},\quad F_{44}=F_{55}=\sum Q_{44}^{(k)}\,\frac{\Delta(z^5)_k}{5}\]

Thermal Resultants

Thermal force, moment and higher-order moment
\[N_{xx}^T = N_{yy}^T = \sum_{k=1}^{N_L}\int_{z_{bot}}^{z_{top}}(Q_{11}+Q_{12})\,\alpha\,\Delta T\;dz\] \[M_{xx}^T = M_{yy}^T = \sum_{k=1}^{N_L}\int_{z_{bot}}^{z_{top}}(Q_{11}+Q_{12})\,\alpha\,\Delta T\cdot z\;dz\] \[P_{xx}^T = P_{yy}^T = \sum_{k=1}^{N_L}\int_{z_{bot}}^{z_{top}}(Q_{11}+Q_{12})\,\alpha\,\Delta T\cdot z^3\;dz\]

§6 — Static Condensation (Quasi-3D) SADECE Q3D

Parameters_stretch.m — This is the key difference from HSDT

The condition \(\sigma_z = 0\) is applied to the 3D constitutive equations, and \(\varepsilon_z\) is mathematically eliminated. This process replaces all stiffness matrices with their "condensed" counterparts.

Step 1 — Inverse Operator (Schur Complement)

Determinant and inverse operator of the 3D stiffness matrix
\[\det K = A_{11}^{3D}\cdot D_{11}^{3D} - \left(B_{11}^{3D}\right)^2\] \[I_{11} = \frac{D_{11}^{3D}}{\det K},\qquad I_{12} = I_{21} = \frac{-B_{11}^{3D}}{\det K},\qquad I_{22} = \frac{A_{11}^{3D}}{\det K}\]

Here \(A_{11}^{3D}=\int C_{11}\,dz\), \(B_{11}^{3D}=\int C_{11}\,z\,dz\), and \(D_{11}^{3D}=\int C_{11}\,z^2\,dz\) (from the \(C_{ij}\) integrals in §5).

Step 2 — Condensed A Matrix

\[A_{11}^c = A_{11}^{3D} - A_{12}^{3D}\!\left(I_{11}\,A_{12}^{3D} + I_{12}\,B_{12}^{3D}\right) - B_{12}^{3D}\!\left(I_{21}\,A_{12}^{3D} + I_{22}\,B_{12}^{3D}\right)\] \[A_{12}^c = A_{12}^{3D} - A_{12}^{3D}\!\left(I_{11}\,A_{12}^{3D} + I_{12}\,B_{12}^{3D}\right) - B_{12}^{3D}\!\left(I_{21}\,A_{12}^{3D} + I_{22}\,B_{12}^{3D}\right)\] \[A_{22}^c = A_{11}^c,\quad A_{21}^c = A_{12}^c\]

Step 3 — Condensed B Matrix

\[B_{11}^c = B_{11}^{3D} - A_{12}^{3D}\!\left(I_{11}\,B_{12}^{3D} + I_{12}\,D_{12}^{3D}\right) - B_{12}^{3D}\!\left(I_{21}\,B_{12}^{3D} + I_{22}\,D_{12}^{3D}\right)\] \[B_{12}^c = B_{12}^{3D} - A_{12}^{3D}\!\left(I_{11}\,B_{12}^{3D} + I_{12}\,D_{12}^{3D}\right) - B_{12}^{3D}\!\left(I_{21}\,B_{12}^{3D} + I_{22}\,D_{12}^{3D}\right)\]

Step 4 — Condensed D Matrix

\[D_{11}^c = D_{11}^{3D} - B_{12}^{3D}\!\left(I_{11}\,B_{12}^{3D} + I_{12}\,D_{12}^{3D}\right) - D_{12}^{3D}\!\left(I_{21}\,B_{12}^{3D} + I_{22}\,D_{12}^{3D}\right)\] \[D_{12}^c = D_{12}^{3D} - B_{12}^{3D}\!\left(I_{11}\,B_{12}^{3D} + I_{12}\,D_{12}^{3D}\right) - D_{12}^{3D}\!\left(I_{21}\,B_{12}^{3D} + I_{22}\,D_{12}^{3D}\right)\]

Step 5 — Condensed E, F, H Matrices

\[E_{11}^c = E_{11}^{3D} - A_{12}^{3D}\!\left(I_{11}\,E_{12}^{3D} + I_{12}\,F_{12}^{3D}\right) - B_{12}^{3D}\!\left(I_{21}\,E_{12}^{3D} + I_{22}\,F_{12}^{3D}\right)\] \[F_{11}^c = F_{11}^{3D} - B_{12}^{3D}\!\left(I_{11}\,E_{12}^{3D} + I_{12}\,F_{12}^{3D}\right) - D_{12}^{3D}\!\left(I_{21}\,E_{12}^{3D} + I_{22}\,F_{12}^{3D}\right)\] \[H_{11}^c = H_{11}^{3D} - E_{12}^{3D}\!\left(I_{11}\,E_{12}^{3D} + I_{12}\,F_{12}^{3D}\right) - F_{12}^{3D}\!\left(I_{21}\,E_{12}^{3D} + I_{22}\,F_{12}^{3D}\right)\]

The \({}_{12}\) and \({}_{21}\) elements for each matrix are calculated using the same formula.

Step 6 — Thermal Force Correction

\[N_{xx}^{T,c} = N_{xx}^T - A_{12}^{3D}\!\left(I_{11}\,N_{zz}^T\right) - B_{12}^{3D}\!\left(I_{22}\,N_{zz}^T\right)\] \[N_{yy}^{T,c} = N_{yy}^T - A_{12}^{3D}\!\left(I_{11}\,N_{zz}^T\right) - B_{12}^{3D}\!\left(I_{22}\,N_{zz}^T\right)\]

The condensed \(A^c, B^c, D^c, E^c, F^c, H^c\) matrices replace the original \(A, B, D, E, F, H\) matrices and are used in all subsequent equations from §7 onwards.

§6.5 — Governing Differential Equations (Equilibrium and Compatibility)

Equilibrium and Geometrical Compatibility Equations Based on HSDT Plate Theory

Using Reddy's HSDT displacement field and the principle of virtual work, the nonlinear governing equations of motion (equilibrium equations) for the FG-GOEAM plate are expressed as:
In-plane force equilibrium equations
\[\frac{\partial N_{xx}}{\partial x} + \frac{\partial N_{xy}}{\partial y} = 0\] \[\frac{\partial N_{xy}}{\partial x} + \frac{\partial N_{yy}}{\partial y} = 0\]
Rotational (moment) equilibrium equations
\[\frac{\partial M_{xx}}{\partial x} + \frac{\partial M_{xy}}{\partial y} - Q_x - c_1 \left(\frac{\partial P_{xx}}{\partial x} + \frac{\partial P_{xy}}{\partial y}\right) + 3c_1 R_x = 0\] \[\frac{\partial M_{xy}}{\partial x} + \frac{\partial M_{yy}}{\partial y} - Q_y - c_1 \left(\frac{\partial P_{xy}}{\partial x} + \frac{\partial P_{yy}}{\partial y}\right) + 3c_1 R_y = 0\]
Transverse (out-of-plane) equilibrium equation
\[\left(N_{xx} + N_{xx}^T\right)\frac{\partial^2 w_0}{\partial x^2} + 2N_{xy}\frac{\partial^2 w_0}{\partial x \partial y} + \left(N_{yy} + N_{yy}^T\right)\frac{\partial^2 w_0}{\partial y^2} + \frac{\partial^2 M_{xx}}{\partial x^2} + 2\frac{\partial^2 M_{xy}}{\partial x \partial y} + \frac{\partial^2 M_{yy}}{\partial y^2} - c_1\left(\frac{\partial^2 P_{xx}}{\partial x^2} + 2\frac{\partial^2 P_{xy}}{\partial x \partial y} + \frac{\partial^2 P_{yy}}{\partial y^2}\right) + q - q_f = 0\]
where \(q\) is the transverse mechanical load (equal to \(0\) in this postbuckling study), and \(q_f\) is the reaction of the two-parameter Winkler-Pasternak elastic foundation:
Elastic Foundation Reaction Equation
\[q_f = K_1 w_0 - K_2 \nabla^2 w_0 = K_1 w_0 - K_2 \left(\frac{\partial^2 w_0}{\partial x^2} + \frac{\partial^2 w_0}{\partial y^2}\right)\]
To satisfy the in-plane force equilibrium equations identically, the Airy stress function \(f(x, y)\) is introduced:
Airy Stress Function Definition
\[N_{xx} = \frac{\partial^2 f}{\partial y^2}, \qquad N_{yy} = \frac{\partial^2 f}{\partial x^2}, \qquad N_{xy} = -\frac{\partial^2 f}{\partial x \partial y}\]
The geometrical compatibility relation of strains is written in terms of the mid-plane strain components and curvatures as:
Strain Compatibility Relation
\[\frac{\partial^2 \varepsilon_{x}^0}{\partial y^2} + \frac{\partial^2 \varepsilon_{y}^0}{\partial x^2} - \frac{\partial^2 \gamma_{xy}^0}{\partial x \partial y} = \left(\frac{\partial^2 w_0}{\partial x \partial y}\right)^2 - \frac{\partial^2 w_0}{\partial x^2}\frac{\partial^2 w_0}{\partial y^2}\]
Substituting the inverted constitutive relations into the strain compatibility equation yields the compatibility equation in terms of the stress function \(f\), deflection \(w_0\), and rotation functions (\(\phi_x, \phi_y\)):
Compatibility Equation in terms of Airy Stress Function
\[J_{11} \frac{\partial^4 f}{\partial x^4} + J_{12} \frac{\partial^4 f}{\partial x^2 \partial y^2} + J_{13} \frac{\partial^4 f}{\partial y^4} = \left(\frac{\partial^2 w_0}{\partial x \partial y}\right)^2 - \frac{\partial^2 w_0}{\partial x^2}\frac{\partial^2 w_0}{\partial y^2} + J_{21} \frac{\partial^4 w_0}{\partial x^4} + J_{22} \frac{\partial^4 w_0}{\partial x^2 \partial y^2} + J_{23} \frac{\partial^4 w_0}{\partial y^4} + J_{31} \frac{\partial^3 \phi_x}{\partial x^3} + J_{32} \left(\frac{\partial^3 \phi_x}{\partial x \partial y^2} + \frac{\partial^3 \phi_y}{\partial x^2 \partial y}\right) + J_{33} \frac{\partial^3 \phi_y}{\partial y^3}\]

§7 — Inverted Stiffness \(C_{ij}\) Coefficients (Denk. 19)

Parameters_Analytical_Solution.m · Parameters_stretch.m

Denominator (determinant-like)
\[\text{Denom} = A_{12}\,A_{21} - A_{11}\,A_{22}\]

First row coefficients

\[C_{11} = \frac{A_{12}}{\text{Denom}},\qquad C_{12} = -\frac{A_{22}}{\text{Denom}}\] \[C_{13} = -\frac{-A_{22}B_{11} + A_{12}B_{21}}{\text{Denom}},\qquad C_{14} = -\frac{-A_{22}B_{12} + A_{12}B_{22}}{\text{Denom}}\] \[C_{15} = -\frac{-A_{22}E_{11} + A_{12}E_{21}}{\text{Denom}},\qquad C_{16} = -\frac{-A_{22}E_{12} + A_{12}E_{22}}{\text{Denom}}\]

Second row coefficients

\[C_{21} = \frac{A_{11}}{-\text{Denom}},\qquad C_{22} = -\frac{A_{21}}{-\text{Denom}}\] \[C_{23} = -\frac{-A_{21}B_{11} + A_{11}B_{21}}{-\text{Denom}},\qquad C_{24} = -\frac{-A_{21}B_{12} + A_{11}B_{22}}{-\text{Denom}}\] \[C_{25} = -\frac{-A_{21}E_{11} + A_{11}E_{21}}{-\text{Denom}},\qquad C_{26} = -\frac{-A_{21}E_{12} + A_{11}E_{22}}{-\text{Denom}}\]

Shear coefficients

\[C_{31} = -\frac{1}{A_{66}},\qquad C_{32} = -\frac{B_{66}}{A_{66}},\qquad C_{33} = -\frac{E_{66}}{A_{66}}\]

§9 — \(S_{ij}\) Coefficients (Appendix A)

Parameters_Analytical_Solution.m · Parameters_stretch.m

Coefficients in the Galerkin integral of the governing equations. \(c_2 = 3c_1 = 4/h^2\).

S1x — Plate stiffness coefficients

§14 — Buckling Loads and Postbuckling Equilibrium (Denk. 38–39)

Parameters_Analytical_Solution.m · Parameters_stretch.m

The result of the entire formulation: critical buckling load \(\lambda_0\) and postbuckling equilibrium path coefficients \(\lambda_1, \lambda_2\).

Common factor

\[\mathcal{T} = \frac{1}{\chi_1\,V_{18} + \chi_2\,V_{19}}\]

Here \(\chi_1 = -1\) (x-direction load), \(\chi_2 = -1\) (biaxial) or \(\chi_2 = 0\) (uniaxial).

Critical buckling load \(\lambda_0\)

\[\lambda_0 = -\left(S_2 + V_{11} + P_{11}V_{14} + P_{21}V_{15}\right)\cdot\mathcal{T}\]
Dimensionless critical load
\[\bar{N}_{cr} = \lambda_0 \cdot \frac{a^2}{E_m\,h^3}\]

Postbuckling linear coefficient \(\lambda_1\)

\[\lambda_1 = -h\left(V_{12} + P_{12}V_{14} + P_{22}V_{15} + P_{11}V_{16} + P_{21}V_{17}\right)\cdot\mathcal{T}\]
For symmetric plates, \(B_{ij}=0 \Rightarrow \lambda_1=0\) (yielding a symmetric postbuckling equilibrium path).

Postbuckling nonlinear coefficient \(\lambda_2\)

\[\lambda_2 = -h^2\left(V_{13} + P_{12}V_{16} + P_{22}V_{17}\right)\cdot\mathcal{T}\]

Complete Postbuckling Equilibrium Path

\[\boxed{\bar{N} = \lambda_0 + \lambda_1\!\left(\frac{W}{h}\right) + \lambda_2\!\left(\frac{W}{h}\right)^2}\]

This equation defines the postbuckling equilibrium path relating the non-dimensional load \(\bar{N}\) to the normalized central deflection amplitude \(W/h\). The difference between HSDT and Quasi-3D is reflected solely in the magnitudes of the \(\lambda_0, \lambda_1, \lambda_2\) coefficients.
