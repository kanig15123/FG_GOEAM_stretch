Physical Rationale & Mechanics

1. Why did Tables 3 & 4 have 0.00% change before?

In the initial thickness stretching implementation, the code only applied static condensation to the membrane (in-plane stretching) stiffnesses (Aij), e.g., A11 = A11 - A13^2/A33. The bending (Dij) and higher-order bending (Fij, Hij) stiffnesses were left completely untouched (uncondensed).

For symmetric plates (like the FG-GOEAM plates in Tables 3 & 4), the material is distributed symmetrically across the mid-plane, resulting in exactly zero bending-stretching coupling (Bij = 0). In linear buckling analysis, the out-of-plane eigenvalue equation is decoupled from the in-plane equations. Since only Aij was relaxed and Bij = 0, the relaxation had no mathematical way to affect the out-of-plane buckling load, leaving λ0 completely invariant (0.0000% change).

2. Why do they change now?

We have implemented a mathematically unified static condensation formulation. Now, all stiffness matrices—specifically bending stiffnesses (Dij) and higher-order stiffnesses (Fij, Hij)—are condensed using the transverse normal stiffness components and coupling terms:

  • D11_c = D11_3D - D12_3D * I22 * D12_3D
  • F11_c = F11_3D - D12_3D * I22 * F12_3D
  • H11_c = H11_3D - F12_3D * I22 * F12_3D
Since the bending stiffnesses that govern the out-of-plane buckling are now condensed, the buckling loads for symmetric plates correctly shift to reflect the thickness stretching effect (varying smoothly up to 9.04%).

3. Table 2 Analysis & Poisson Double-Counting Bug

In asymmetric FGM plates (Table 2), Bij ≠ 0, so bending and stretching are coupled in the linear range. Previously, Table 2 showed a 4.03% decrease in buckling load. However, this was caused by integrating the standard 2D plane-stress stiffnesses Qij (which already condense out σzz = 0 pointwise) and then applying the condensation formula again. This double-counted the transverse Poisson's ratio contraction, making the plate artificially more flexible.

In our corrected formulation, the un-condensed stiffnesses are integrated using 3D elasticity constants Cij instead of Qij. For a homogeneous plate, the condensed stiffnesses simplify exactly to the standard HSDT plane-stress values. For the asymmetric FGM plate, the stretching effect is now correctly resolved to a physically consistent, very small difference (0.0005% change).

Resolved Bugs & Validation Details

The initial zip stretch files had critical discrepancies that corrupted results. Here is how they were resolved:

Graphene Omission

The original stretch code evaluated E_eff as E_metal * fE, omitting graphene's Young's modulus and the Halpin-Tsai mixture rule.

Restored correct mixture equations matching baseline.

Symmetry Typos

Integral calculations in the layers loop used Q11 stiffness components instead of Q12 for coupling terms a21, b21, d21, etc.

Corrected loop indexes to match exact matrix symmetry.

Integral Formulation

Integrals for E, F, H used f(z) instead of z^n. Since parameters solver already multiplied these by HSDT shape terms, lambda0 fell to 0.40.

Modified integrals to use standard z-power integrals (z^3, z^4, z^6).
