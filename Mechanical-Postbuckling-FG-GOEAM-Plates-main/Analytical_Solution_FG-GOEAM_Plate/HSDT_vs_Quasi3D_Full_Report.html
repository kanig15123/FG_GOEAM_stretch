Theory Overview

Both theories are built on the same displacement field and governing equations, but differ in one fundamental assumption: whether the transverse normal strain \(\varepsilon_z\) is zero (HSDT) or non-zero and accounted for (Quasi-3D).

Standard Theory
HSDT — Higher-order Shear Deformation Theory
Assumes plane-stress condition: the transverse normal stress \(\sigma_z = 0\), which also forces \(\varepsilon_z = 0\) in an ideal sense. Stiffness coefficients are built from 2D reduced stiffnesses \(Q_{ij}\): \[Q_{11} = \frac{E}{1-\nu^2}, \quad Q_{12} = \frac{E\nu}{1-\nu^2}, \quad Q_{66} = \frac{E}{2(1+\nu)}\] The bending-stretching matrices \(A_{ij}, B_{ij}, D_{ij}, E_{ij}, F_{ij}, H_{ij}\) are all computed from \(Q_{ij}\) integrated through thickness.
VS
Quasi-3D Theory
Quasi-3D — Full Thickness Stretching
Allows \(\varepsilon_z \neq 0\). Uses full 3D elasticity constants \(C_{ij}\): \[C_{11} = \frac{E(1-\nu)}{(1+\nu)(1-2\nu)}, \quad C_{12} = \frac{E\nu}{(1+\nu)(1-2\nu)}\] The condition \(\sigma_z = 0\) is then imposed through static condensation, which modifies all stiffness matrices \(A_{ij}^c, B_{ij}^c, D_{ij}^c, \ldots\) to account for Poisson effects in the thickness direction.

HSDT Displacement Field

Eq. (1) — Reddy-type HSDT
\[u = u_0 - z\,w_{,x} + c_1 z^3\!\left(w_{,x} + \phi_x\right)\] \[v = v_0 - z\,w_{,y} + c_1 z^3\!\left(w_{,y} + \phi_y\right)\] \[w = w_0 \qquad(\varepsilon_z = 0)\]

where \(c_1 = 4/(3h^2)\), \(\phi_x, \phi_y\) are rotations, and \(w_0\) is the transverse displacement. Thickness stretching is absent — \(w\) is independent of \(z\).

Quasi-3D Displacement Field

Eq. (1) — Enhanced with thickness stretching
\[u = u_0 - z\,w_{,x} + c_1 z^3\!\left(w_{,x} + \phi_x\right)\] \[v = v_0 - z\,w_{,y} + c_1 z^3\!\left(w_{,y} + \phi_y\right)\] \[w = w_0 + \xi(z)\,\psi \qquad(\varepsilon_z \neq 0)\]

The transverse displacement now includes a thickness-stretching function \(\xi(z)\) and amplitude \(\psi\). This introduces a 6th DOF and a non-zero \(\varepsilon_z = d\xi/dz \cdot \psi\).

HSDT Constitutive Law

2D Plane-Stress Reduced Stiffness
\[\begin{bmatrix}\sigma_x\\\sigma_y\\\tau_{xy}\end{bmatrix} = \begin{bmatrix}Q_{11}&Q_{12}&0\\Q_{12}&Q_{11}&0\\0&0&Q_{66}\end{bmatrix}\begin{bmatrix}\varepsilon_x\\\varepsilon_y\\\gamma_{xy}\end{bmatrix}\] \[Q_{11}=\frac{E}{1-\nu^2},\quad Q_{12}=\frac{E\nu}{1-\nu^2},\quad Q_{66}=\frac{E}{2(1+\nu)}\]

The denominator \(1-\nu^2\) accounts for plane-stress Poisson coupling only in the in-plane directions.

Quasi-3D Constitutive Law

Full 3D Elasticity Constants
\[\begin{bmatrix}\sigma_x\\\sigma_y\\\sigma_z\\\tau_{xy}\end{bmatrix} = \begin{bmatrix}C_{11}&C_{12}&C_{12}&0\\C_{12}&C_{11}&C_{12}&0\\C_{12}&C_{12}&C_{11}&0\\0&0&0&C_{66}\end{bmatrix}\begin{bmatrix}\varepsilon_x\\\varepsilon_y\\\varepsilon_z\\\gamma_{xy}\end{bmatrix}\] \[C_{11}=\frac{E(1-\nu)}{(1+\nu)(1-2\nu)},\quad C_{12}=\frac{E\nu}{(1+\nu)(1-2\nu)}\]

The denominator \((1+\nu)(1-2\nu)\) accounts for full 3D Poisson coupling including through-thickness (\(z\)-direction) effects.

Stiffness Matrix Definitions (Common to Both Theories)

Eq. (17) — Through-thickness integration
\[(A_{ij},\,B_{ij},\,D_{ij},\,E_{ij},\,F_{ij},\,H_{ij}) = \int_{-h/2}^{h/2} \bar{Q}_{ij}\,(1,\,z,\,z^2,\,z^3,\,z^4,\,z^6)\,dz\]

For HSDT: \(\bar{Q}_{ij} = Q_{ij}(z)\).   For Quasi-3D: \(\bar{Q}_{ij}\) are replaced by the condensed \(Q_{ij}^c\) derived from \(C_{ij}\) via static condensation (see next section).

🔧
Static Condensation — The Key Difference

The Quasi-3D modification is implemented through static condensation: the condition \(\sigma_z = 0\) is imposed on the full 3D constitutive law, eliminating \(\varepsilon_z\) in terms of \(\varepsilon_x\) and \(\varepsilon_y\), then substituting back. This modifies all stiffness matrices.

1
Write the full 3D stress-strain relation for \(\sigma_z\)

From the 3D elasticity law, the thickness stress is:

\[\sigma_z = C_{12}\,\varepsilon_x + C_{12}\,\varepsilon_y + C_{11}\,\varepsilon_z = 0\]
2
Solve for the thickness strain \(\varepsilon_z\)
\[\varepsilon_z = -\frac{C_{12}}{C_{11}}\,(\varepsilon_x + \varepsilon_y)\]

This shows that the Quasi-3D theory accounts for the thickness contraction/expansion due to in-plane strains (Poisson effect in z-direction).

3
Substitute back into the in-plane stress equations
\[\sigma_x = \left(C_{11} - \frac{C_{12}^2}{C_{11}}\right)\varepsilon_x + \left(C_{12} - \frac{C_{12}^2}{C_{11}}\right)\varepsilon_y\]

The condensed in-plane modulus is: \(\;Q_{11}^c = C_{11} - C_{12}^2/C_{11}\)

4
Generalized condensation for heterogeneous plates (with z-coupling)

For a plate where all stiffness components vary through thickness, the condensation is the full Schur complement. Define the thickness-direction inverse operator:

\[\det K = A_{11}^{3D} \cdot D_{11}^{3D} - \left(B_{11}^{3D}\right)^2\] \[I_{11} = \frac{D_{11}^{3D}}{\det K},\quad I_{12} = \frac{-B_{11}^{3D}}{\det K},\quad I_{22} = \frac{A_{11}^{3D}}{\det K}\]

where \(A_{11}^{3D} = \int C_{11}\,dz\), \(B_{11}^{3D} = \int C_{11}\,z\,dz\), \(D_{11}^{3D} = \int C_{11}\,z^2\,dz\).

5
Condensed stiffness matrices

Each stiffness matrix is condensed by subtracting the coupling contribution:

\[A_{11}^c = A_{11}^{3D} - A_{12}^{3D}\left(I_{11}\,A_{12}^{3D} + I_{12}\,B_{12}^{3D}\right) - B_{12}^{3D}\left(I_{21}\,A_{12}^{3D} + I_{22}\,B_{12}^{3D}\right)\] \[D_{11}^c = D_{11}^{3D} - B_{12}^{3D}\left(I_{11}\,B_{12}^{3D} + I_{12}\,D_{12}^{3D}\right) - D_{12}^{3D}\left(I_{21}\,B_{12}^{3D} + I_{22}\,D_{12}^{3D}\right)\] \[H_{11}^c = H_{11}^{3D} - E_{12}^{3D}\left(I_{11}\,E_{12}^{3D} + I_{12}\,F_{12}^{3D}\right) - F_{12}^{3D}\left(I_{21}\,E_{12}^{3D} + I_{22}\,F_{12}^{3D}\right)\]

Similarly for \(B_{ij}^c\), \(E_{ij}^c\), \(F_{ij}^c\). These condensed matrices replace their HSDT counterparts in all governing equations.

6
Thermal force correction
\[N_{xx}^T{}^c = N_{xx}^T - A_{12}^{3D}\left(I_{11}\,N_{zz}^T\right) - B_{12}^{3D}\left(I_{22}\,N_{zz}^T\right)\]

The thermal resultant also requires correction because the thickness thermal stress \(N_{zz}^T\) couples back into in-plane forces.

KEY DIFFERENCE SUMMARY

  • HSDT uses \(Q_{ij}\) with denominator \((1-\nu^2)\) — only 2D Poisson coupling
  • Quasi-3D uses \(C_{ij}\) with denominator \((1+\nu)(1-2\nu)\) — full 3D Poisson coupling
  • For \(\nu = 0.3\): \(Q_{11} = E/0.91\) while \(C_{11} = 0.7E/0.28 = 2.5E\) — large raw difference
  • After condensation: condensed \(A_{11}^c = h \cdot (C_{11}-C_{12}^2/C_{11})\) — difference from HSDT depends on non-uniformity of material through \(z\)
  • For uniform (U-WGr) material: condensed stiffness = HSDT stiffness exactly (see proof below)
  • For non-uniform (X-WGr, O-WGr): differences arise from z-gradient of \(C_{ij}\), up to ~9%

Governing Buckling Equations

The critical buckling load and postbuckling coefficients are obtained from the Galerkin method applied to the governing equations. All expressions below use the condensed stiffnesses for Quasi-3D, or the original HSDT stiffnesses for the standard theory.

Inverted Stiffness Coefficients (Eq. 19)

\[C_{11} = \frac{A_{12}}{A_{12}A_{21}-A_{11}A_{22}}, \quad C_{12} = -\frac{A_{22}}{A_{12}A_{21}-A_{11}A_{22}}\] \[C_{13} = -\frac{-A_{22}B_{11}+A_{12}B_{21}}{A_{12}A_{21}-A_{11}A_{22}}, \quad C_{31} = -\frac{1}{A_{66}}\] \[C_{32} = -\frac{B_{66}}{A_{66}}, \quad C_{33} = -\frac{E_{66}}{A_{66}}\]

Geometric Stress-Function Coefficients (Eq. 21)

\[J_{11}=C_{21},\quad J_{12}=C_{11}+C_{22}-C_{31},\quad J_{13}=C_{12}\] \[J_{21}=C_{23}-c_1C_{25},\quad J_{22}=C_{13}-c_1C_{15}-C_{32}+c_1C_{33}\] \[J_{31}=-c_1C_{25},\quad J_{32}=-c_1(C_{15}+C_{26})+2c_1C_{33}\]

Critical Buckling Load (Eq. 38)

Non-dimensional critical buckling load \(\bar{N}_{cr}\)
\[\lambda_0 = (-1)\cdot\frac{S_2 + V_{11} + P_{11}V_{14} + P_{21}V_{15}}{\chi_1 V_{18} + \chi_2 V_{19}}\] \[\bar{N}_{cr} = \lambda_0 \cdot \frac{a^2}{E_m h^3}\]

where \(S_2\) is the foundation contribution, \(V_{ij}\) are Galerkin integral coefficients, \(P_{ij}\) are postbuckling amplitude coefficients from static condensation of the rotational DOFs, and \(\chi_1, \chi_2\) define the biaxial/uniaxial loading ratio.

Postbuckling Equilibrium Path (Eq. 38–39)

\[\bar{N} = \lambda_0 + \lambda_1\left(\frac{W}{h}\right) + \lambda_2\left(\frac{W}{h}\right)^2\]
λ₀ — Critical Buckling
Onset of buckling. Depends on all stiffness matrices. Quasi-3D always ≥ HSDT.
λ₁ — Linear Postbuckling
Linear slope of load-deflection curve. Zero for symmetric plates due to B₁₂=0.
λ₂ — Nonlinear Postbuckling
Quadratic load stiffening/softening. Governs how rapidly the plate re-stiffens.

Analytical Proof: Why U-WGr Shows ~0% Difference

For the uniform graphene distribution (U-WGr), material properties are z-independent constants. This leads to an exact algebraic cancellation showing that the condensed stiffness equals the HSDT stiffness.

THEOREM — U-WGr Invariance
\[\text{For uniform } C_{ij}(z) = \text{const}:\quad A_{11}^c = h\left(C_{11} - \frac{C_{12}^2}{C_{11}}\right) = h\cdot Q_{11} = A_{11}^{HSDT}\]

PROOF

For z-independent material: \(A_{11}^{3D} = C_{11}h\), \(A_{12}^{3D} = C_{12}h\), \(B_{11}^{3D} = B_{12}^{3D} = 0\) (symmetry), \(D_{11}^{3D} = C_{11}h^3/12\).

The condensed membrane stiffness:

\[A_{11}^c = A_{11}^{3D} - \frac{(A_{12}^{3D})^2}{A_{11}^{3D}} = C_{11}h - \frac{(C_{12}h)^2}{C_{11}h} = h\left(C_{11} - \frac{C_{12}^2}{C_{11}}\right)\]

Substituting \(C_{11} = \frac{E(1-\nu)}{(1+\nu)(1-2\nu)}\) and \(C_{12} = \frac{E\nu}{(1+\nu)(1-2\nu)}\):

\[C_{11} - \frac{C_{12}^2}{C_{11}} = \frac{E(1-\nu)}{(1+\nu)(1-2\nu)}\left[1 - \frac{\nu^2}{(1-\nu)^2}\right] = \frac{E(1-\nu)}{(1+\nu)(1-2\nu)}\cdot\frac{(1-\nu)^2-\nu^2}{(1-\nu)^2}\] \[= \frac{E}{(1+\nu)(1-2\nu)}\cdot\frac{1-2\nu}{1-\nu} \cdot \frac{1-\nu}{1} = \frac{E}{(1+\nu)(1-\nu)} = \frac{E}{1-\nu^2} = Q_{11} \quad\blacksquare\]

The same cancellation holds for all higher-order terms \(D_{11}^c, F_{11}^c, H_{11}^c\), proving that for U-WGr the Quasi-3D theory reduces exactly to HSDT — hence the observed ~0.000% difference in all U-WGr results.
